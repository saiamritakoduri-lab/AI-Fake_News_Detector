# AI-Fake_News_Detector
AI Fake News Detector is a machine learning–based system that detects whether a news article is authentic or fake. With the rapid spread of misinformation on the internet and social media, this project aims to help users quickly verify the credibility of news content.

The application uses Natural Language Processing (NLP) to clean and transform text data, followed by a supervised machine learning model trained on labeled news datasets. The model learns patterns in writing style, word usage, and contextual features that commonly appear in fake news.

Users can enter a news headline or full article, and the system will analyze the text and provide a prediction indicating whether the news is likely real or fake.
